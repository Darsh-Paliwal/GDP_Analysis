## 📌 Project Overview
This project analyzes **Global GDP data from 1960 to 2016** using **Python, Pandas, Matplotlib, and Plotly**. The goal of this analysis is to explore **economic growth patterns of countries**, compare GDP trends, and visualize GDP growth over time.

Using **Exploratory Data Analysis (EDA)** and **interactive visualizations**, this project uncovers insights about:

- GDP growth of different countries
- Comparison of GDP between multiple countries
- Fastest growing economies
- Global GDP trends

---

# 📊 Dataset Information
The dataset contains GDP values for **256 countries and regions** from **1960 to 2016**.

### Dataset Columns

| Column | Description |
|------|-------------|
| Country Name | Name of the country |
| Country Code | Country abbreviation |
| Year | Year of the GDP record |
| Value | GDP value in USD |

Example:

| Country Name | Year | GDP |
|--------------|------|-----|
| India | 1960 | 3.70e10 |
| China | 1960 | 5.97e10 |
| World | 1960 | 1.36e12 |

# 🎯 Project Objectives

This project answers several key analytical questions:

1. How many countries are present in the dataset?
2. How many years of GDP data are available for India?
3. What was the **World GDP in 1969**?
4. What was the **GDP growth percentage of India in 1994 compared to 1993**?

Additional analysis tasks include:

- Calculating **GDP growth for every country**
- Identifying **top fastest growing economies**
- Visualizing **GDP trends**
- Creating **interactive country comparisons**

# 🔍 Key Insights

- India shows **steady GDP growth especially after the 1990 economic reforms**
- China demonstrates **rapid GDP expansion after 2000**
- Global GDP has **consistently increased over time**
- Some GDP growth outliers occur due to **missing data in early years**
